# TerraCycle

A sustainability project

We Compare. We Collect. We Connect. 
TerraCycle is a frontend web app that makes recycling rewarding. Users schedule waste pickups 
or drop-offs, earn TerraCredits for every kilogram recycled, track their personal environmental 
impact, and connect with a local sustainability community.

File Structure 
1.  landing page.html     (Marketing/home page) 
2.  login page.html       (Login with email + password) 
3. signup page.html       (Create a new account) 
4. dashboard.html         (User dashboard- stats, achievements, activity feed) 
5. community.html         (Community & Challenges- recipes, meetups, upcycling) 
6. feed.html              (Eco content feed- blogs and upcycling tutorials)         
7. schedule.html          (Pick between bag pickup or drop-off)     
8. pickup.html            (Schedule a bag pickup form) 
9. dropped.html           (Schedule a bag drop-off form) 
10. redeem.html           (Redeem TerraCredits via scratch cards) 
11. help.html             (Waste routing guide- vendors by waste type (Delhi NCR))        

Features
1. Landing Page — Hero section, 3-step how-it-works, community highlights, dashboard 
feature previews 
2. Auth — Login and signup pages with basic JS validation, routes to dashboard on 
success 
3. Dashboard: Personalized welcome, quick stats (pickups, kg recycled, CO₂ averted, 
trees saved, fuel saved), achievements, recent activity, community impact, and smart 
waste segregation guide 
4. Community & Challenges: Category cards for Recipes, Local Meetups, and Upcycling; 
links to feed 
5. Feed: Eco tips and upcycling tutorials (e.g. t-shirts into tote bags) with images and step 
breakdowns 
6. Schedule: Choose between bag pickup or drop-off, then redirects to respective form 
page 
7. Pickup / Drop-off Forms: Select date, phone number, location (Amsterdam / Delhi / 
Mumbai), time slot, and waste type (Plastic, Metal, Cardboard, Paper) 
8. Redeem: View current TerraCredits, cashback won, TerraSCORE, scratch card vouchers 
(1 point = ₹1) 
9. Help / Waste Routing: Shows where each waste type goes; lists real Delhi NCR vendors 
for dry waste (JAAGRUTI, Econscious, WMARS, IPCA) and organic waste (EcoPolluTech, 
Saahas, Rise Foundation, DPGS); lets users select a vendor

Tech Stack 
Structure:  HTML5 
Styling:    CSS3- Flexbox, Grid, CSS custom properties 
Fonts:      Google Fonts- Inter, DM Sans, Fraunces 
Icons:      Font Awesome 6 
Animations: CSS keyframes (fadeUp, fadeDown) 
Interactivity: Vanilla JavaScript (form validation, page routing)

Running Locally 
No installs needed — it's all static HTML. 
User flow: landing page.html → login page.html → dashboard.html → schedule.html → 
pickup.html or dropped.html → redeem.html
